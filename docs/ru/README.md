# Заметки для владельца проекта

Это AI harness для форка `nyr/openvpn-install`.

## Как использовать

1. Распакуй архив в корень форка.
2. Передай Codex содержимое `plan.md`.
3. Попроси Codex работать фазами и обновлять execution plan.
4. Проверяй, что Codex запускает:

```bash
bash -n openvpn-install.sh
./run-tests.sh
```

Тесты будут красными на исходном upstream-скрипте. Это нормально: они описывают целевое состояние после модификации.
